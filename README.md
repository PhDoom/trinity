# Trinity Continuum

Unofficial Foundry Virtual Tabletop implementation of the Trinity Continuum Core RPG, by Onyx Path Publishing. The system currently supports Trinity Core, Adventure, Aeon, Aberrant, Anima, Aether, Aegis and Assassins.

## Info

This system was built from RandomCards original system. Full disclosure and transparency as I know there are strong feelings towards AI. I'm not a coder and used Gemini AI to update the system to Foundry v14 and expanded it, meaning that Gemini wrote the code.

## How to Use

Install system in Foundry with: https://raw.githubusercontent.com/PhDoom/trinity/master/system.json

Due to IP concerns, this system does not include any actual content. It does include Attributes, Skills and items, but does not have any descriptions for those items. Items can be created and added to character sheet directly or via drag-and-dropping pre-made items. Items include:
* Paths
* Contacts
* Edges
* Skill Tricks
* Gifts
* Equipment
* Powers
* Weapons
* Vehicles

Information on each of these elements can be found in the Trinity Continuum Core Rulebook, and other Trinity products. Generally, you would populate these items from the rulebook(s), then drop the items on the character sheet. 
The system has a dice pool roller and an easy method for setting success and critical/exploding dice levels.
